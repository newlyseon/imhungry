# 단식 앱 User Flow

## 개요

사용자가 앱에 진입하면 현재 상황을 질문하고, 대화형 스텝으로 단식을 설정하는 온디맨드 모델.

### 상태 전환 구조

```
Idle (대기) ──→ 타이머 (단식 중) ──→ 완료 ──→ Idle
     │                                         ↑
     └─→ 예약 대기 ──→ 타이머 ──→ 완료 ────────┘
```

---

## Step 1: 온보딩 진입

```
[화면] 상황 선택

  안녕하세요! 잘 오셨어요. 시작해볼까요?

  a. 바로 단식을 시작할 수 있어요
  b. 오늘 식사가 예정되어 있어요
```

→ a 선택 시: Flow A
→ b 선택 시: Flow B

---

## Flow A: 바로 단식을 시작할 수 있어요

### Step 2: 마지막 식사 시간 확인

```
[화면] 시간 확인

  마지막 식사를 마친 시간은요?

  a. 방금 먹었어요 → (현재 시각 자동 입력)
  b. 시간 직접 선택 → 시간 피커
```

### Step 3: 단식 패턴 선택

**"방금 먹었어요" 선택 시 (예: 오후 8시)**

```
[화면] 패턴 선택

  지금부터 단식을 시작할게요!
  마지막 식사: 오후 8시 기준

  a. 16:8  → 내일 오후 12시에 다시 식사 가능
  b. 14:10 → 내일 오전 10시에 다시 식사 가능
  c. 12:12 → 내일 오전 8시에 다시 식사 가능
```

**"시간 직접 선택" 시 (예: 어제 오후 7시)**

```
[화면] 패턴 선택

  이미 13시간째 단식 중이에요! 💪
  마지막 식사: 어제 오후 7시 기준

  a. 16:8  → 오전 11시에 식사 가능 (3시간 남음!)
  b. 14:10 → 오전 9시에 식사 가능 (1시간 남음!)
  c. 12:12 → 지금 바로 식사 가능! ✅
```

### Step 4: 타이머 시작

**"방금 먹었어요" → 16:8 선택 시**

```
[화면] 타이머 홈

  🌙 단식 중
  내일 오후 12시에 식사 가능

  16:00 카운트다운 시작
  진행률 0%
```

**"시간 직접 선택" → 16:8 선택 시**

```
[화면] 타이머 홈

  🌙 단식 중
  오전 11시에 식사 가능

  03:00 카운트다운
  진행률 81%
```

---

## Flow B: 오늘 식사가 예정되어 있어요

### Step 2: 식사 종료 시간 설정

```
[화면] 시간 선택

  오늘 마지막 식사는 몇 시쯤 끝날까요?

  * 시간 피커 (예: 오후 1시, 오후 2시 ...)
  * 또는 빠른 선택: 점심만 / 저녁까지
```

→ "오후 1시" 선택

### Step 3: 단식 패턴 선택

```
[화면] 패턴 선택

  단식을 예약할게요!
  마지막 식사 종료: 오후 1시 기준

  a. 16:8  → 내일 오전 5시에 다시 식사 가능
  b. 14:10 → 내일 오전 3시에 다시 식사 가능
  c. 12:12 → 내일 오전 1시에 다시 식사 가능
```

→ "16:8" 선택

### Step 4: 예약 확인

```
[화면] 예약 확인

  ✅ 단식이 예약됐어요!

  식사 종료 예정: 오늘 오후 1시
  단식 시작: 오후 1시 이후 자동 시작
  다음 식사: 내일 오전 5시

  [식사 종료 시간 변경]
```

→ 홈: 예약 대기 상태

### 예약 시간 도달 후

```
[시스템] 오후 1시 도달
         → 푸시: "단식이 시작됐어요!"
         → 홈 자동 전환: 타이머 화면
```

---

## 단식 진행 중 재진입

```
1. [진입] 앱 오픈
2. [시스템] 단식 진행 중 감지 → 자동으로 타이머 홈 표시
3. [화면] 타이머 홈
         - "식사까지 10분!"
         - 진행률 99%
4. [분기] 사용자 선택
         ├─ 참는다 → 앱 닫기 → 알림: "식사 가능 시간이에요 🍽️"
         └─ 포기 → "지금 종료" → 기록 저장 → Idle 홈 복귀
```

---

## 단식 완료 후 재진입

```
1. [시스템] 단식 시간 종료 → 푸시: "단식 완료! 🎉"
2. [진입] 앱 오픈
3. [화면] 완료 화면
         - "16시간 2분 달성!"
         - 연속 기록 표시
         - 간단 기록 입력 (컨디션, 메모 — 선택)
4. [액션] 기록 저장 또는 스킵
5. [화면] Idle 홈 복귀
         - 마지막 기록 요약 카드
         - "지금 어떤 상황인가요?" 다시 표시
```

## 홈 화면 상태 요약

| 상태 | 진입 조건 | 주요 정보 | CTA |
|------|----------|----------|-----|
| **Idle** | 단식 없음 | Step 1 선택지 (a/b) | 상황 선택 |
| **예약 대기** | Flow B 완료 | 예약 정보 + 식사 종료 시간 | "시간 변경" / "취소" |
| **타이머** | 단식 진행 중 | 남은 시간 + 식사 가능 시각 | "단식 종료" |
| **완료** | 단식 시간 종료 | 달성 시간 + 연속 기록 | 기록 저장 → Idle |
